# 1. CPU占用率曲线
```C++
#include<iostream>
#include<windows.h>
using namespace std;

int main() {
	for (;;) {
		for (int i = 0; i < 96000000; i++);
		Sleep(10);
	}
	system("pause");
	return 0;
}
```
